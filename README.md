# WebAnalyzer

> X-ray any website: detect its tech stack, hosting, DNS, security headers, and WHOIS — instantly, privately, all in one click.

See what any website is really built with — in one click.

WebAnalyzer is the inspection tool for curious developers, agencies, and site owners who want to look under the hood of any website. Open it on any page and instantly reveal the technologies, hosting, and infrastructure powering it.

## What you get

### 🔍 Detected Stack

Identify 60+ technologies at a glance — CMS platforms (WordPress, Drupal, Shopify), JavaScript frameworks (React, Next.js, Vue), CSS frameworks, web servers, backend languages, CDNs, analytics, and more. Every detection shows a confidence score and the exact evidence behind it, so you always know **why** something was flagged.

### 🌐 Network Intelligence

Go beyond the surface with full DNS records (A, AAAA, MX, TXT, CAA, and more), passive subdomain discovery via Certificate Transparency logs, IP geolocation with country flags, and CDN/WAF detection (Cloudflare, Sucuri, Akamai, and others).

### 🛡️ Security Headers

Audit a site's HTTP security headers at a glance. WebAnalyzer sorts them into three groups — **configured**, **recommended (missing)**, and **misconfigured** — covering HSTS, Content-Security-Policy, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy. Each finding explains what the header does, what's wrong, and how to fix it.

### 📋 WHOIS Lookup

Pull registration details in a clean, readable layout — registrar, creation and expiry dates, status, and contact information where public records allow.

### ⚡ One-Click DNS Export

Migrating to Cloudflare? Export DNS records as a ready-to-import BIND zone file — correctly formatted, so the import just works.

### 📸 Instant Screenshots

Capture the full analysis as an image and copy it to your clipboard or save it as PNG — perfect for reports, audits, and sharing with clients.

### 🌎 Automatic Localization

The interface follows your browser's language automatically, with full English and Brazilian Portuguese (pt-BR) translations. Any unsupported language falls back to English.

## Built on principle

- ✅ **100% free** — no paywalls, no premium tier, no catch.
- ✅ **Private by design** — no tracking, no analytics, no telemetry. Your browsing is never sent to us, because we have no servers collecting it.
- ✅ **Fully open source** — every line of code is public and auditable. Trust through transparency, not promises.
- ✅ **No account needed** — install and use immediately.

WebAnalyzer reads only public technical signals — HTTP headers, public page markup, and cookie names (never their values). It never touches your passwords, form data, or personal information.

## Privacy

WebAnalyzer has no backend and collects nothing. DNS, WHOIS, subdomain, and geolocation lookups are made directly from your browser to public services, and results are cached locally. See [PRIVACY.md](PRIVACY.md) for details.
